# Portfolio — Behavioral Reconstructibility Cases

This portfolio contains public-facing and semi-public case material derived from internal reconstructive audits.

The portfolio does **not** store raw reconstructive engine artifacts.

Raw artifacts belong in:

```text
/work_structure/09_reconstructive_engine/
```

Portfolio material contains:

- public case summaries
- evidence-safe snapshots
- branch summaries
- conflict summaries
- reconstructibility findings
- observability gaps
- portfolio narratives
- implementation notes

## Rule

Do not copy raw evidence registries, full conflict logs, frozen topology graphs, or internal arbitration overlays into this folder unless they have been explicitly sanitized.
