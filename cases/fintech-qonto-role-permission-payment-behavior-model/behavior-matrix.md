# Behavior Matrix

| Area | Action | Condition | Expected behavior | Friction |
|---|---|---|---|---|
| Owner/Admin | Change role | Full authority | Admin access granted | Authorization path unclear |
| Manager | Approve/pay | Permission-dependent | Action allowed within scope | Approval ambiguity |
| Employee | Submit request | Limited access | Request sent for approval | Blocked approval path |
| Incoming SDD | Mandate pending | Client approval required | Collection waits | Expired/cancelled link |
| Incoming SDD | Mandate signed | Eligible account | Collection processed | Refund/hold window |
| Profile update | Phone/email change | SCA or fallback | Identity verified | Recovery friction |
