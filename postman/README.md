# Postman Assets

This directory is reserved for public-safe Postman assets related to documentation behavior analysis.

It may contain:

- sanitized collections
- anonymized environments
- public API examples
- mock requests
- OpenAPI-related exploration files

Do not include:

- API keys
- bearer tokens
- cookies
- private workspace data
- personal credentials
- production endpoints with credentials
- non-public client material
