# Vale Rules — Portfolio Layer

These notes define writing constraints for portfolio-facing material.

## Avoid

- generic clarity claims
- unsourced recommendations
- invented behavior
- artificial workflow completion
- global system claims
- marketing language

## Prefer

- scope-local statements
- evidence-safe wording
- branch-aware descriptions
- visible uncertainty
- documented limits
- operational consequences

## Required phrasing discipline

Use:

- “The available evidence supports…”
- “Within this scope…”
- “This branch remains fragile because…”
- “The documentation does not currently expose…”
- “This requires human validation…”

Avoid:

- “The system does…”
- “Users can…”
- “The workflow is…”
- “The documentation should define the observable behavior…”
